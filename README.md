# 装修小白工具箱

一个无需注册、数据保存在浏览器本地的装修管理工具，可用于预算、流程、工人、沟通和材料记录。

## 在线体验

[www.xyzz2508.xyz](http://www.xyzz2508.xyz)

## 隐私说明

- 装修数据使用浏览器 `localStorage` 保存在当前设备，不会由本站上传到服务器。
- 清除浏览器数据、无痕模式结束或更换域名后，本地数据可能丢失。
- 请定期使用页面中的“备份我的数据”导出 JSON 文件。
- 导入和导出均在浏览器内完成，请自行妥善保管备份文件。

## 本地运行

下载仓库后，在仓库目录启动任意静态文件服务器，例如：

```bash
python -m http.server 8000
```

然后打开 `http://localhost:8000`。不要直接双击 HTML 后判断全部功能是否正常，使用本地静态服务器更可靠。

## 目录结构

- `index.html`：页面入口
- `css/`：页面样式
- `js/`：业务逻辑与本地数据管理
- `vendor/`：Chart.js、SheetJS 等第三方本地依赖
- `images/`：公开展示素材
